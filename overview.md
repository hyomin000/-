0. 프로젝트 개요

1.프로젝트명

사용자 맞춤형 피싱 탐지 시스템 및 IoT 경고 디바이스 (부제: 가족 안심 원격 알림 서비스)

2.개발 배경

보이스피싱과 스미싱 수법이 날로 지능화되면서, 이제 피싱은 특정 연령층만의 문제가 아니라 스마트폰을 사용하는 우리 모두의 일상을 위협함.
하지만 기존 보안 앱들은 텍스트 위주의 복잡한 UI를 가지고 있어, 긴급한 상황에서 직관적으로 위험을 인지하기 어려움.
특히 디지털 소외계층은 알림을 보고도 적절한 대응을 하지 못하는 경우가 많음.

4.개발 목적
모든 사용자가 피싱 위험을 즉각적으로 감지할 수 있도록 돕는 범용 보안 시스템 구축을 목적
특히,디지털 기기 조작에 익숙하지 않은 사용자들을 위해 아두이노 기반의 물리적 경고 디바이스를 부가 서비스로 제공하여 시청각적인 직관성을 극대화
+위험 상황 발생 시 보호자의 기기로 실시간 알림을 공유하는 연동 시스템을 구현하여 2중 보안망을 형성하고자 함

5.핵심 구현 기능
지능형 문장 분석 엔진:입력된 텍스트의 키워드와 패턴을 분석하여 피싱 위험도를 계산합니다.


기본 자바 기반:GUI를 통한 분석 결과 출력

부가 서비스:아두이노와 연동하여 LED와 부저로 물리적 경고 전달

보호자 연동 안심 서비스: 사용자 본인뿐만 아니라, 연동된 보호자의 기기에도 위험 상황을 즉시 공유하여 범죄 예방의 골든타임을 확보
